# Requisitos previos para jugar en red

* Un dispositivo con conexión inalámbrica.
* Que *Configuración de red* > *Activar Wi-Fi* esté activado. Los dispositivos pueden conectarse al mismo punto de acceso o a ninguno y usar redes ad hoc para conectarse directamente.
* Que *Ajustes de Juegos* > *Ajustes de Netplay* > *Activar Netplay* esté activado.
* Si se usa la red ad hoc, activa también *Automatically Use Hotspot for Local Netplay* y, opcionalmente, *Automatically Create Lobby*.

# Para ser el anfitrión de una partida de Netplay:

* Mantén presionado el Botón Este (:material-gamepad-circle-right:) en un juego compatible de la lista de juegos y selecciona *Iniciar Partida en Netplay*.
* Luego, selecciona *Hospedar Partida en Netplay*. El juego se iniciará y configurará el host de Netplay.

# Para unirte a una partida de Netplay:

* Ejecuta el mismo juego (preferiblemente la misma ROM).
* Presiona ++"Function"++ + el Botón Sur (:material-gamepad-circle-down:) para acceder al menú de Retroarch.
* Presiona el Botón Sur (:material-gamepad-circle-down:) para subir un nivel en el menú y luego baja hasta *Juego en red*.
* Selecciona *Actualizar lista de servidores de juego en red* y presiona el Botón Este (:material-gamepad-circle-right:) para buscar juegos.
* Selecciona el host de la lista y presiona el Botón Este (:material-gamepad-circle-right:) para unirte a la partida.
* Si el host no aparece, revisa la configuración del wifi y asegúrate de que los dispositivos estén en el mismo punto de acceso. En el caso de la conexión ad hoc, en algunas circunstancias podrías tener que conectarte manualmente al punto de acceso ad hoc iniciado por el host antes de ejecutar el juego.

Puedes encontrar más parámetros de configuración, como el nombre y el puerto del punto de acceso ad hoc, en el archivo `batocera.conf`.
