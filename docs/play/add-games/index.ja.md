# :material-layers-plus: KNULLIへのゲームの追加

KNULLIにはゲームを追加するためのオプションがいくつかあります。お使いのデバイスの性能に応じて、どのオプションを選択するかを決定する必要があります。ゲームを追加する前に、ゲームやその他のユーザーデータのためのストレージ容量を確保しておくことをお勧めします。

## ゲームを追加する前に

* [ゲームストレージ](game-storage.md) - KNULLIのフォルダ構造について学びましょう。
* [SDカードのフォーマット](formatting.md) - フォーマットオプションについて学びましょう。
* [2枚目のSDカードの設定](second-sd-card.md) - シングルSDカードとデュアルSDカードの設定について学びましょう。

## ゲームストレージの設定が完了したら

* [ネットワーク転送](network-transfer.md) - Wi-Fi経由でゲームを追加する方法を学びましょう。
* [USB経由のアクセス](accessing-via-usb.md) - USB経由でデバイスにアクセスする方法を学びましょう。
* [SDカードへの直接アクセス](accessing-sd-card.md) - WindowsからSDカードに直接アクセスする方法を学びましょう(該当する場合)。
