# :material-micro-sd: 2枚目のSDカードを使用する

!!! info "デュアルSDカード構成への移行"

    このセクションは、**KNULLIの初期設定を行っている**ユーザー向けです。すでにシングルSDカードでKNULLIの設定を開始していて、**デュアルSDカード構成に移行したい**場合は、代わりに[デュアルSDカード移行ガイド](../../guides/dual-sd-card-migration.md)を参照してください。

2つ目のSDカードスロットを備えたデバイスでKNULLIを使用する場合、2つ目のスロットをセカンダリSDカードに使用できます。セカンダリSDカードは、プライマリSDカードの*SHARE*パーティションの**代わりに**使用できます。

SDカード1とSDカード2を**同時**にゲームストレージとして使用することは**できません**のでご注意ください。SD1**もしくは**SD2の**どちらか一方**しか同時に使用できません。詳細については[ゲームストレージ](./game-storage.md)セクションを参照してください。

デュアルSDカード構成には、依然として多くの利点があります。:

* **アップデートに失敗した場合**、CFW全体を再度フラッシュする必要が生じることがあります。デュアルSD構成の場合、SD1を再フラッシュするだけで済み、システムデータ以外はSD2に安全に保存されているため、削除されることがありません。
* もしあなたが複数のデバイスを持っている場合、データ用SDカードを**複数のデバイスで**使用することができます。
* **バックアップを作成する**のは、SDカード全体を複製するだけで済むため、より簡単になるかもしれません。例えば、イメージを作成してDSカードに書き込むといった方法です。

## 2枚目のSDカードの設定

* デバイスの電源が入っている場合、オフにします。
* 電源オフの間に2枚目のSDカードを挿入します。
* KNULLIを起動し、++"Start"++ボタンを押して*システム設定*を選択肢ます。
* *ストレージ*セクションの中の*ストレージデバイス*を選択してください。
    * *内蔵*("内蔵"は、システムSDの*SHARE*パーティションを意味します。)から、2枚目のSDカードの名前*例)SHARE - 25.6G*を選択します。
* KNULLIを再起動し、++"Start"++ボタンを押して*終了*メニューから*再起動*を選択します。
* もしあなたのデバイスがまだあなたの選択したファイルシステムで**フォーマットされていない場合**
    * [フォーマット](./formatting.md)セクションに従って、**KNULLI内蔵フォーマッタ**を使用してフォーマットしてください。
    * 適用するために再起動してください。
* 再起動中、KNULLIは自動的に2枚目のSDカードの中に[userdataフォルダ](./game-storage.md)にあるフォルダとファイルを/*SHARE*パーティションに生成します。
* 2枚目のSDカードがexFAT形式でフォーマットされている場合は、デバイスの電源を切り、カードを取り出してください。カードをコンピュータに挿入すれば、そのままデータにアクセスできます。
