Compatibility issues with revisioned Anbernic RG XX models!

Please be aware that since the release of KNULLI Gladiator II, Anbernic has released and shipped revisioned models of some of their RG XX series devices. If you attempt to run KNULLI Gladiator II on a revisioned device, KNULLI will simply not boot.

Luckily, we are aware of the problem and we are currently running a public alpha test with builds that should be compatible with the new device models.

If you have problems installing KNULLI on a brand new RG XX device, either wait for the next public release of KNULLI or join us on Discord and type !revisions in the help channel to get access to the public alpha builds and help us test them.
